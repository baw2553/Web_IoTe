หน้าlogin มี 2 roles ได้แก่ 1.Admin 2.Member
โดย :
วิธีการเข้าหน้า Admin - ต้องใช้อีเมลของAdminโดยเฉพาะ 
(อีเมลของAdmin ได้แก่ pikulkaew.ta@kmitl.ac.th ,67010411@kmitl.ac.th)
เมื่อเข้าสู่ระบบของAdmin ในส่วนของฟังก์ชัน Threads จะสามารถเพิ่มและยกเลิกข้อความกระทู้ของทุกๆบุคคลได้

วิธีการเข้าหน้า Member - สามารถใช้ได้ทุกอีเมลยกเว้นอีเมลของAdmin 
เมื่อเข้าสู่ระบบของMember ในส่วนของฟังก์ชัน Threads จะสามารถเพิ่มและยกเลิกข้อความกระทู้ของตนเองได้เท่านั้น

และเมื่อต้องการเข้าฟังก์ชัน IoTE Runner 
จำเป็นต้องใช้อีเมลในการเข้าสู่ระบบเพื่อบันทึกสถิติ โดยทุกๆอีเมลสามารถเข้าสู่ระบบของฟังก์ชันนี้ได้

Data Base ที่มีได้แก่ :
1. Comment - ใช้ในส่วนของฟังก์ชัน Threads เพื่อแสดงความคิดเห็นต่างๆ
2. Topic - หัวข้อกระทู้ในส่วนของฟังก์ชัน Threads
3. Leader Board - ใช้ในส่วนของฟังก์ชัน IoTE Runner เพื่อบันทึกสถิติ
